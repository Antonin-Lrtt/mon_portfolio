Projet 3



Portfolio



Description du projet


Ce projet consiste à créer son site web qui consistera à la réalistion d'un Portfolio qui portera sur le sujet qu l'on souhaite et qui répondera aux attentes demandées.


Objectifs pédagogiques


  • Utiliser le CSS pour la mise en forme du contenu.

  • Publier le site sur GitHub Pages.

  • Apprendre à organiser un site web avec plusieurs pages liées.


Compétences visées


  • Structurer et organiser un site web en utilisant le HTML sémantique et les bonnes pratiques de codage.

  • Mettre en forme et personnaliser l’apparence d’un site avec le CSS

  • Créer un portfolio complet avec une navigation cohérente.

  • Présenter et valoriser son travail avec une identité graphique cohérente


Résultat obtenu


Le projet se compose d'une page d'accueil, d'une page projets et d'une page par projet. Le site est responsive et conforme aux standards du W3C
